Функции
Объявление

var square = function(x) {
  return x * x;
};

function square(x) {
  return x * x;
}

Вызов

console.log(square(12));

Области видимости - переменные и функции, объявленные внутри функции (а не любого блока) являются локальными и снаружи не видны.
Локальные переменные для блоков можно создавать с помощью "let".

Функцию можно переопределять как переменную.
Можно объявлять функцию после вызова.

Помещать объявление функций стоит только в код основной программы или в другие функции . Внутрь условных операторов помещать не стоит.

Стек вызова функций - компьютер запоминает контекст, откуда произошел вызов функции, чтобы потом вернуть управление в нужное место.
Место где комп. запоминает контекст называется стеком. Хранение стека требует памяти.

Необязательные аргументы: если передать лишние - функция их не будет использовать, если передать недостаточно, то недостающие будут проинициализированы как undefined.

Замыкание - это мясо. Поскольку функции можно рассматривать как переменные, можно определять вложенные функций как локальные переменные, а потом можно работать со ссылкой на экземпляр локальной переменной (а в данном случае - функции)

function multiplier(factor) {
  return function(number) {
    return number * factor;
  };
}

var twice = multiplier(2);
var thrice= multiplier(3);

console.log(thrice(5));
// → 15

console.log(twice(5));
// → 10

Хороший вариант мысленной модели – представлять, что функция замораживает код в своём теле и обёртывает его в упаковку. Когда вы видите return function(...) {...}, представляйте, что это пульт управления куском кода, замороженным слоями для употребления позже. Каждое присвоение дает доступ к следующему слою. "twice=multiplier(2);" фиксирует верхний слой и дает обращаться к следующему.

"Не беспокойтесь насчёт быстродействия, пока вы точно не уверены, что программа тормозит."

Выращивание функций - постепенная декомпозиция задачи до разумных элементов, оформленных как функции, избегая излишней универсализации, стараясь не превышать универсальность, необходимую для решаемых задач.
