Автоматическое преобразование типов
0, NaN и пустая строка “”  - это false для булевых значений
для проверки абсолютной идентичности использовать "===" и "!==" (полезно чтобы избежать косяков из-за автоматического приведения типов)

Короткое вычисление логических операторов.
Обязательно вычисляется только левая часть, остальное по необходимости.

console.log(null || "user")
// → user
console.log("Karl" || "user")
// → Karl

Оператор || как откат к значению по умолчанию. Если выражение слева можно преобразовать в true то вернет его, если нет - то вернет правое выражение. С && - наоборот.
